simpleSnake.jar can be run both with or without arguments and is played using the arrow keys.

If the game is run without arguments, then they playable grid is 10 by 10.
If it is run with a single argument then the grid is args[0] by 10.
And lastly if run with 2 arguments then the grid is args[0] by args[1].

The first argument is the horizontal size (sizeX), while the second is the vertical size (sizeY).
To run the game use this command:
    java -jar simpleSnake.jar (sizeX) (sizeY)
